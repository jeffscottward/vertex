# Audio Assets

Place electronic music tracks here for gameplay. Recommended sources for royalty-free/Creative Commons electronic music:

## Free Music Sources

1. **Free Music Archive** - https://freemusicarchive.org/genre/Electronic/
   - Look for CC BY or CC0 licensed tracks
   - Electronic, Ambient, Trance genres

2. **Incompetech (Kevin MacLeod)** - https://incompetech.com/music/
   - Large library of royalty-free tracks
   - Attribution required (CC BY 3.0)

3. **Patrick de Arteaga** - https://patrickdearteaga.com/
   - OGG format, loop-ready
   - Great for game development

4. **OpenGameArt** - https://opengameart.org/art-search-advanced?field_art_type_tid[]=12
   - Community-contributed game music
   - Various CC licenses

5. **SampleSwap** - https://sampleswap.org/mp3/creative-commons/free-music.php
   - 320kbps MP3s
   - Free for non-commercial use

## Recommended Tracks for REZ-style Game

Look for tracks with:
- Strong, consistent beat (120-140 BPM ideal)
- Electronic/Trance/Synthwave style
- Clear bass drops for beat detection
- 2-4 minute length for level tracks

## File Format

- MP3 or OGG preferred
- 128-320 kbps quality
- Name files like: `level1.mp3`, `level2.mp3`, etc.

## Example Attribution

When using CC BY tracks, add to your README:
```
Music: "Track Name" by Artist Name
Licensed under CC BY 3.0
https://creativecommons.org/licenses/by/3.0/
```
